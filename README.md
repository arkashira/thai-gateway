# 🚀 thai-gateway

**Power businesses with seamless Thai-language API connectivity.** thai-gateway is an open-source API gateway designed to simplify communication between Thai-language applications and external services.

## Why thai-gateway?

* **🔒 Secure**: Built with Rust for robust security and performance.
* **🚀 Scalable**: Designed to handle high traffic and large-scale deployments.
* **🌎 Global**: Supports Thai-language APIs for international businesses.
* **📈 Analytics**: Provides detailed metrics and analytics for API performance.
* **🔍 Debuggable**: Includes a built-in debugger for easy issue resolution.
* **📚 Documented**: Well-documented codebase for easy understanding and maintenance.
* **🤝 Community-driven**: Open-source and community-driven for continuous improvement.

## Feature Overview

| Feature | Description |
| --- | --- |
| API Routing | Route Thai-language APIs with ease |
| Authentication | Secure API access with authentication |
| Rate Limiting | Prevent abuse with rate limiting |
| Analytics | Monitor API performance with detailed metrics |
| Debugging | Easily debug issues with built-in debugger |

## Tech Stack

* Rust
* Rust std library
* Rust async runtime
* Rust HTTP client
* Rust JSON serialization

## Project Structure

* `app`: Application code
* `business`: Business logic
* `docs`: Documentation
* `gateway`: API gateway code
* `src`: Source code
* `tests`: Unit tests

## Getting Started

1. **Install Rust**: `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
2. **Clone repository**: `git clone https://github.com/thai-gateway/thai-gateway.git`
3. **Build project**: `cargo build`
4. **Run project**: `cargo run`
5. **Test project**: `cargo test`

## Deploy

1. **Deploy to production**: `cargo run --release`
2. **Deploy to staging**: `cargo run --profile staging`
