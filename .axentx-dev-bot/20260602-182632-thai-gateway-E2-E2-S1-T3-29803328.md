## 1. Acceptance Criteria  
- **AC1** – A GET request to `/metrics` returns **HTTP 200**.  
- **AC2** – The response body contains an entry for every AI provider that is configured via environment variables (`PROVIDERS`).  
- **AC3** – Each provider entry includes:  
  - `remaining` – an integer ≥ 0 representing the current remaining quota.  
  - `reset_in_seconds` – an integer ≥ 0 indicating the seconds until the rate‑limit window resets.  
- **AC4** – When a provider’s `remaining` count is **0**, the entry is still present with `remaining: 0` and the correct `reset_in_seconds`.  
- **AC5** – The metrics are exposed in **JSON** when `METRICS_FORMAT=json` (default) and in **Prometheus exposition format** when `METRICS_FORMAT=prometheus`.  
- **AC6** – The endpoint is protected by the same API‑key middleware used by the gateway; a request without a valid key returns **HTTP 401**.  
- **AC7** – The data returned reflects the *current* state of the in‑memory rate‑limit tracker; a change in the tracker is immediately visible on the next request.
