**Final Solution – Add a production‑ready Prometheus metrics layer to the gateway**

The reviewer’s only blocker is the **absence of a `/metrics` endpoint** that emits the three metric families the dashboard expects:

| Metric family (required by the dashboard) | Meaning |
|-------------------------------------------|---------|
| `gateway_request_latency_seconds`        | Histogram of request latency, labelled by `provider` and HTTP `status` |
| `gateway_request_errors_total`           | Counter of error responses (status ≥ 400), labelled by `provider` |
| `gateway_provider_requests_total`        | Counter of how many requests are forwarded to each upstream AI provider |

All three can be supplied with a **single, self‑contained change** that:

1. **Creates the metric objects and registers them** with the default Prometheus registry.  
2. **Installs a Gin middleware** that records latency and errors for every request and updates the counters.  
3. **Exposes the `/metrics` HTTP handler** on the existing Gin engine.  
4. **Provides a tiny helper** (`RecordProviderRequest`) that the code which talks to the upstream provider can call to bump the provider‑specific counter.

The implementation touches only the API layer, adds a new file, a one‑line call in the router‑initialisation, and a `go.mod` entry. No external services, no schema changes, and no runtime configuration are required. The whole change can be written, compiled and verified in **≈ 2 hours**.

    )

func resetMetrics() {
    prometheus.DefaultRegisterer = prometheus.NewRegistry()
}

// Test that /metrics returns proper content‑type and status
func TestMetricsEndpoint_StatusAndContentType(t *testing.T) {
    resetMetrics()
    registerTestMetrics()

    req := httptest.NewRequest(http.MethodGet, "/metrics", nil)
    w := httptest.NewRecorder()
    promhttp.Handler().ServeHTTP(w, req)

    resp := w.Result()
    assert.Equal(t, http.StatusOK, resp.StatusCode)
    assert.Equal(t, "text/plain; version=0.0.4; charset=utf-8", resp.Header.Get("Content-Type"))
}

// Test that required metric families are present
