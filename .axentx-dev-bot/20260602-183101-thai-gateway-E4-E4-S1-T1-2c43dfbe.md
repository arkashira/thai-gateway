```python
from prometheus_client import start_http_server, Gauge, Counter, Histogram
import time

# Define Prometheus metrics
TOTAL_REQUESTS = Counter('total_requests', 'Total number of requests')
SUCCESSFUL_REQUESTS = Counter('successful_requests', 'Number of successful requests')
FAILED_REQUESTS = Counter('failed_requests', 'Number of failed requests')
LATENCY = Histogram('request_latency_seconds', 'Request latency in seconds', buckets=(1, 2, 5, 10))
RATE_LIMIT_USAGE = Gauge('rate_limit_usage', 'Provider-specific rate limit usage', ['provider'])

def initialize_prometheus():
    # Start Prometheus HTTP server on port 8000
    start_http_server(8000)

def record_request(success=True):
    TOTAL_REQUESTS.inc()
    if success:
        SUCCESSFUL_REQUESTS.inc()
    else:
        FAILED_REQUESTS.inc()

def record_latency(latency):
    LATENCY.observe(latency)

def update_rate_limit(provider, usage):
    RATE_LIMIT_USAGE.labels(provider=provider).set(usage)

# Example usage
if __name__ == "__main__":
    initialize_prometheus()
    while True:
        record_request(success=True)
        record_latency(time.time())
        update_rate_limit('provider1', 0.8)
        time.sleep(1)
```

### 1. Acceptance Criteria
- **AC1** – `GET /metrics` returns HTTP 200 within 200 ms.
- **AC2** – Response body is in Prometheus exposition format and includes the following metric families:
  - `gateway_requests_total` (counter) with labels `status` (`success`, `failure`) and `provider`.
  - `gateway_request_latency_seconds` (summary) exposing `p90` and `p99` quantiles.
- **AC3** – For each configured provider, a metric `gateway_provider_rate_limit_used` (gauge) is emitted showing current rate‑limit consumption.
- **AC4** – No metric contains raw user‑provided data (e.g., request prompts, personal identifiers). A static regex scan of the payload must find zero matches.
- **AC5** – Metrics are updated in real time: a request sent to the gateway is reflected in the counters/latency within 1 second.
- **AC6** – The endpoint is protected by the existing gateway authentication middleware (i.e., unauthenticated requests receive HTTP 401).
