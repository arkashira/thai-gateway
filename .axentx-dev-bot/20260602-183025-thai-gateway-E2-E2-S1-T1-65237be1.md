## 1. Acceptance Criteria  

| # | Criterion | Measurable Target |
|---|-----------|-------------------|
| 1 | **Latency** | Translation request completes in < 0.5 s (measured from start of `translate()` call to return). |
| 2 | **Quality** | Pilot Likert scores (1–5) average ≥ 4.0 across 100 sampled translations. |
| 3 | **Optionality** | Header `X-Translate: false` bypasses translation and forwards original prompt unchanged. |
| 4 | **Fallback** | If translation model fails, gateway returns original prompt with HTTP 200 and logs warning. |
| 5 | **Thread‑Safety** | Multiple concurrent translation requests do not corrupt shared state or exceed memory limits. |
| 6 | **Error Handling** | Non‑JSON or malformed requests return HTTP 400 with clear error message. |
| 7 | **Logging** | Each translation event logs request ID, latency, and success flag. |

#[cfg(test)]
mod tests {
    use super::*;
    use std::time::Instant;

    // Helper to create a mock translator
    fn mock_translator() -> Translator {
        Translator::new(MockModel::new())
    }

    #[test]
    fn test_translate_latency() {
        let translator = mock_translator();
        let prompt = "Hello, world!";
        let start = Instant::now();
        let _ = translator.translate(prompt).unwrap();
        let elapsed = start.elapsed();
        assert!(elapsed.as_millis() < 500, "Latency > 0.5s");
    }

    #[test]
    fn test_translate_quality() {
        let translator = mock_translator();
        let prompt = "Good morning";
        let translated = translator.translate(prompt).unwrap();
        // MockModel returns deterministic translation
        assert_eq!(translated, "สวัสดีตอนเช้า");
    }

    #[test]
    fn test_optional_translation_header() {
        let mut req = Request::new();
        req.headers_mut().insert("X-Translate", "false".parse().unwrap());
        let response = handle_request(req);
        assert_eq!(response.body, original_prompt());
    }

    #[test]
    fn test_missing_header_defaults_to_translate() {
        let mut req = Request::new();
        let response = handle_request(req);
        assert_ne!(response.body, original_prompt());
    }

    #[test]
    fn test_translation_fallback_on_error() {
        let translator = Translator::new(ErrorModel::new());
        let result = translator.translate("test");
        assert!(result.is_ok());
        assert_eq!(result.unwrap(), "test"); // original prompt returned
    }

    fn test_invalid_json_returns_400() {
        let req = Request::new_with_body("{invalid json");

        assert_eq!(resp.status, StatusCode::BAD_REQUEST);
    }
}
