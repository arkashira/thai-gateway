## 1. Acceptance Criteria  

1. **Quota Persistence** – For each provider, the gateway must store a numeric `remaining_quota` value in Redis, keyed by a deterministic string (e.g., `quota:{provider_id}`) with a TTL that matches the provider’s quota window.  
2. **Quota Decrement** – Every successful request to a provider decrements its `remaining_quota` atomically in Redis.  
3. **Quota Check** – Before routing a request, the gateway must read the provider’s `remaining_quota`; if it is `> 0`, the request proceeds.  
4. **Provider Rotation** – If the current provider’s quota is `0`, the gateway must select the next provider in the configured rotation order and retry the quota check.  
5. **All Providers Exhausted** – When all providers have `remaining_quota == 0`, the gateway must return an HTTP `429 Too Many Requests` with a `Retry-After` header set to the nearest upcoming TTL among all providers.  
6. **Atomicity & Race‑Safety** – Quota decrement and read operations must be atomic to avoid race conditions under concurrent requests.  
7. **Graceful Degradation** – If Redis is unavailable, the gateway should fall back to in‑memory quota tracking for the current request cycle and log a warning.

## 2. Unit Tests (pseudo‑code, Jest)

```js
// quota.test.js
const { QuotaStore } = require('./quota');
const redisMock = require('redis-mock');

describe('QuotaStore', () => {
  let quota;
  let client;

  beforeAll(() => {
    client = redisMock.createClient();
    quota = new QuotaStore(client);
  });

  afterEach(() => client.flushall());

  test('stores quota with TTL', async () => {
    await quota.setQuota('providerA', 100, 60); // 60s TTL
    const ttl = await client.ttl('quota:providerA');
    expect(ttl).toBeLessThanOrEqual(60);
    const val = await client.get('quota:providerA');
    expect(val).toBe('100');
  });

  test('decrements quota atomically', async () => {
    await quota.setQuota('providerB', 5, 120);
    await quota.decrement('providerB');
    const val = await client.get('quota:providerB');
    expect(val).toBe('4');
  });

  test('returns null if provider not found', async () => {
    const val = await quota.getQuota('unknown');
    expect(val).toBeNull();
  });

  test('handles concurrent decrements correctly', async () => {
    await quota.setQuota('providerC', 2, 30);
    await Promise.all([
      quota.decrement('providerC'),
      quota.decrement('providerC'),
      quota.decrement('providerC'), // should not go below 0
    ]);
    const val = await client.get('quota:providerC');
    expect(val).toBe('0');
  });
});
```
