# thai-gateway
axentx product · A lightweight API gateway that balances rate limits across multiple AI provider APIs while translating prompts into Thai, tailored for mid-sized Thai e-commerce platforms.

## Onboarding Guide

### Registering Your Platform

To register your platform and obtain an API key, follow these steps:

1. **Make an initial registration request**:
   Send a POST request to the registration endpoint. This will automatically generate an API key for your platform.
   
   ```bash
   curl -X POST https://api.axentx.com/thai-gateway/register \
     -H "Content-Type: application/json" \
     -d '{"platform_name": "YourPlatformName"}'
   ```

2. **Retrieve your API Key**:
   Upon successful registration, you will receive a JSON response containing your unique API key. Store this key securely as it grants access to the gateway services.
   
   ```json
   {
     "api_key": "your_unique_api_key_here"
   }
   ```

### API Key Scoping

Each API key is strictly scoped to the platform it was generated for. This ensures that there is no unauthorized cross-platform access, thereby maintaining the security and integrity of each platform's data and operations.

### Rate Limit Enforcement

Rate limits are enforced individually for each platform based on the API key used. This prevents any single platform from overwhelming the system resources and ensures fair usage across all registered platforms.

### Complete Onboarding Flow

1. **Register your platform** as described above to get your API key.
2. **Securely store the API key** for future use in making requests to the gateway.
3. **Start utilizing the gateway services** by including your API key in the headers of your requests.
   
   Example request:
   ```bash
   curl -X GET https://api.axentx.com/thai-gateway/service-endpoint \
     -H "Authorization: Bearer your_unique_api_key_here"
   ```

By following these steps, you can seamlessly onboard your Thai e-commerce platform to the axentx thai-gateway and start leveraging its capabilities without any setup complications.
```
