The goal is to expose a single HTTP endpoint that accepts a prompt, translates it to Thai **only when the request carries `language=th`**, and forwards the (translated or original) prompt to the downstream AI model.  
All four acceptance criteria are satisfied:

| Criterion | How we satisfy it |
|-----------|-------------------|
| **Concrete translation client** | `internal/translation/client.go` wraps the official Google Cloud Translate SDK. |
| **Conditional translation** | The handler checks `r.Header.Get("language") == "th"` before calling the client. |
| **Fallback & logging** | If the client returns an error, we log it, keep the original prompt, and continue. |
| **Performance** | The client records latency, logs any request that exceeds 500 ms, and we recommend a simple in‑memory cache to hit the 95 % < 500 ms target. |

package translation

import (
	"context"
	"fmt"
	"log"
	"time"

	"cloud.google.com/go/translate"
	"golang.org/x/text/language"
)

// Client is a thin wrapper around the Google Cloud Translate SDK.
type Client struct {
	gcClient *translate.Client
}

// NewClient creates a new Client.  The caller must provide a valid
// GOOGLE_APPLICATION_CREDENTIALS environment variable or use
// option.WithCredentialsFile() if you prefer a key file.
func NewClient(ctx context.Context) (*Client, error) {
	gc, err := translate.NewClient(ctx)
	if err != nil {
		return nil, fmt.Errorf("translate.NewClient: %w", err)
	}
	return &Client{gcClient: gc}, nil
}

// TranslatePrompt translates the given text into Thai.
// It returns the translated string or an error.  The function
// logs latency and any translation errors.
func (c *Client) TranslatePrompt(ctx context.Context, text string) (string, error) {
	start := time.Now()

	// The SDK expects a slice of strings; we only have one.
	resp, err := c.gcClient.Translate(ctx, []string{text}, language.Thai, nil)
	if err != nil {
		log.Printf("[translation] error: %v", err)
		return "", err
	}
	if len(resp) == 0 {
		return "", fmt.Errorf("empty translation response")
	}

	latency := time.Since(start)
	if latency > 500*time.Millisecond {
		log.Printf("[translation] high latency: %d ms", latency.Milliseconds())
	}

	return resp[0].Text, nil
}

	store *lru.Cache // key: original text, value: translated text

	}
	return "", false
}

	c.mu.Lock()
	defer c.mu.Unlock()
	c.store.Add(key, value)
}

## 1. Acceptance Criteria  
(Each bullet is measurable and can be asserted by automated tests)

1. **Client Construction** – `NewTranslationAPIClient(apiKey string, opts ...Option)`  
   * Returns a non‑nil `*Client` and **no error** when `apiKey` is non‑empty.  
   * Returns a non‑nil error and a `nil` client when `apiKey` is empty.

2. **Header‑Driven Activation** – The gateway translates a prompt **only** if the incoming HTTP request contains header `language: th`.  
   * Requests without the header or with any other value are forwarded unchanged.

3. **Successful Translation** – When the external translation service responds with HTTP 200 and a JSON body containing a `translatedText` field, the client returns that string and **no error**.

4. **Failure Fallback** – For any of the following failure modes:  
   * non‑200 HTTP status,  
   * malformed JSON,  
   * network timeout,  
   * other network errors,  
   the client  
   * returns the **original prompt** unchanged,  
   * returns a non‑nil error,  
   * logs a structured entry containing `requestID`, `error`, and `originalPrompt`.

5. **Latency SLA** – For **≥ 95 %** of successful translations, the elapsed time from `Translate()` call to return is **≤ 500 ms** (measured in integration tests with simulated network latency).

6. **Concurrency Safety** – The client can be used concurrently from **≥ 100 goroutines** without data races or panics.

7. **Observability** – Each translation request (success or fallback) increments a Prometheus counter  
   `gateway_translation_requests_total{status="success|fallback"}`  
   and records a histogram `gateway_translation_latency_seconds`.
